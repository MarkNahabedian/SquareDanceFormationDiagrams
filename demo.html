dancers.js os a small JavaScript library for drawing square dance formaqtions in a web browser. Each dancer is represented by a Dancer object. A dancer is located bt an X and a y coordinate expressed in the dancer coordinate system. The dancer coordinate system is scalled such that dancers who are either next to or facing each other have distance 1. Each dancer also has a facing direction. A facing direction of 0 has the dancer's back towards the caller (like couble #1 in a squared set). Each increase by 1 in facimng direction rotates the dancer by one wall to their left, so a facing direction of 1 would have the dqancer facing the same wall as couple #2 in a squared set. Fractional coordinates and directions are allowed.

The cancers are arranged on a dance floor by placing t hem in a Floor object. Each Floor object is associated with an initially empty svg element on the web page. Graphical representations of the dancers will be added to that svg element when the web page is finished loading.

Squared Set
new Floor("squared-set", [
    new Dancer(3, 1, 0, "1", Dancer.gender.GUY),
    new Dancer(2, 1, 0, "1", Dancer.gender.GAL),
    new Dancer(1, 2, 1, "2", Dancer.gender.GUY),
    new Dancer(1, 3, 1, "2", Dancer.gender.GAL),
    new Dancer(2, 4, 2, "3", Dancer.gender.GUY),
    new Dancer(3, 4, 2, "3", Dancer.gender.GAL),
    new Dancer(4, 3, 3, "4", Dancer.gender.GUY),
    new Dancer(4, 2, 3, "4", Dancer.gender.GAL)
  ]);
